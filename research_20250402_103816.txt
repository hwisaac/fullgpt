# XZ Backdoor 리서치 보고서

## 1. 소개
XZ backdoor은 2024년 2월에 Linux 빌드의 xz 유틸리티 내 liblzma 라이브러리 버전 5.6.0 및 5.6.1에 악성 백도어가 도입된 사례입니다.

## 2. Wikipedia 정보
- 악성 백도어는 "Jia Tan"이라는 이름을 사용하는 계정에 의해 도입되었습니다.
- 해당 백도어는 특정 Ed448 개인 키를 보유한 공격자에게 영향을 받은 Linux 시스템에서 OpenSSH를 통해 원격 코드 실행을 제공합니다.
- 이 문제는 CVE-2024-3094 번호를 받았으며, 취약점 및 노출 사항으로 할당되었습니다.

## 3. 추가 정보
더 많은 정보를 얻기 위해 DuckDuckGo를 사용하여 추가 조사를 진행했습니다.

## 4. 결론
XZ backdoor는 Linux 시스템에서의 보안 문제로, 악의적인 공격자가 원격 코드 실행을 허용하는 백도어를 도입한 사례입니다.
